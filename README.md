## âš™ Installation

1. Clone repository

```sh
git clone https://github.com/ulysses-ck/express-react-dynamodb

```

There are two folders with front and backend

```
express-react-dynamodb
|
|---client
|
|---back

```

2. Install dependencies

```sh
cd client
npm install

cd ../back
npm install
```

3. Init backend

```bash
cd client
npm run start
```

4. Init client
```bash
cd back
npm run start
```

6. Build for production(only for client)

```sh
npm run build
```
